# Stride Free UX Wireframe (Mobile First)

## Navigation Shell

- Top: contextual screen header
- Main: card-based content area
- Bottom: thumb-friendly nav tabs
  - Home
  - Track
  - History
  - Analytics
  - Badges
  - Settings

## Screen Blueprint

1. Splash
- Brand mark
- Fast launch transition

2. Onboarding (3 slides)
- Value proposition
- Progress indicators
- Single CTA per step

3. Home Dashboard
- Goal progress rings
- Weekly summary tiles
- Smart recommendation
- Last walk snapshot

4. Live Tracking
- Large pace + key metrics
- Route preview map
- One-tap controls: Start / Pause / Resume / Stop
- Error/status strip for GPS

5. Map View
- Full-height interactive map
- Route replay slider
- Elevation + summary stats

6. Walk Summary
- Session completion hero
- Route card
- Full metrics grid
- Export and share actions

7. History
- Chronological cards
- View, share, delete controls

8. Analytics
- Daily/weekly/monthly trend chart
- Records and rolling totals

9. Achievements
- XP level progress
- Badge list with lock/unlock state

10. Profile/Settings
- Body metrics
- Goals
- Theme/units/notifications
- Optional auth panel
